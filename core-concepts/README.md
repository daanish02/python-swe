# Core Concepts

## About This Section

This section explores intermediate Python concepts that build upon the fundamentals. These topics deepen your understanding of how Python works and introduce powerful features for writing more sophisticated, maintainable code.

You'll learn object-oriented programming, functional programming patterns, decorators, generators, and Python's module system. These concepts form the bridge between basic Python and advanced professional development.

## Contents

### [Comprehensions and Functional Tools](comprehensions-and-functional.md)

List comprehensions, dictionary comprehensions, set comprehensions, generator expressions, and functional programming tools (map, filter, reduce, lambda functions). Writing concise, expressive code.

### [Object-Oriented Programming](object-oriented-programming.md)

Classes and objects, methods and attributes, **init** and other special methods, inheritance, composition, polymorphism, method resolution order (MRO), and designing class hierarchies.

### [Advanced Functions](advanced-functions.md)

Closures, first-class functions, nested functions, function attributes, decorators (function and class decorators), decorator patterns, and functional programming concepts in Python.

### [Iterators and Generators](iterators-and-generators.md)

The iteration protocol, creating iterators with **iter** and **next**, generator functions with yield, generator expressions, lazy evaluation, and when to use generators vs lists.

### [Context Managers](context-managers.md)

Understanding context managers, the with statement, creating custom context managers using **enter** and **exit**, the contextlib module, and resource management patterns.

### [Modules and Packages](modules-and-packages.md)

Deep dive into the import system, package structure, **init**.py files, relative vs absolute imports, namespace packages, and organizing larger projects into packages.
