# Fundamentals

## About This Section

This section covers the core building blocks of Python programming. These are the foundational elements you'll use in every Python program - from basic syntax and data types to functions and file operations. Mastering these fundamentals is essential before moving to more advanced concepts.

The focus is on understanding not just how to use these features, but why they work the way they do and when to apply them effectively.

## Contents

### [Syntax and Data Types](syntax-and-data-types.md)

Variables, basic data types (int, float, str, bool, None), operators (arithmetic, comparison, logical), comments, naming conventions, and basic I/O operations. The foundation of writing any Python code.

### [Control Flow](control-flow.md)

Conditional statements (if/elif/else), loops (for, while), match statements, and control keywords (break, continue, pass). How to control program execution flow and make decisions in code.

### [Data Structures](data-structures.md)

Deep understanding of Python's built-in data structures: lists, tuples, dictionaries, and sets. Their characteristics, operations, performance implications, and when to use each one.

### [Functions](functions.md)

Defining functions, parameters and arguments, return values, scope and namespaces, default arguments, \*args and \*\*kwargs, and function documentation. Building reusable code blocks.

### [File Operations](file-operations.md)

Reading and writing files, working with different file formats (text, CSV, JSON), path handling with pathlib, context managers for file handling, and basic file system operations.

### [Error Handling](error-handling.md)

Understanding exceptions, try/except blocks, catching specific exceptions, finally clause, raising exceptions, and common built-in exceptions. Handling errors gracefully in your programs.

### [Modules and Imports](modules-and-imports.md)

Using modules, import statements, import variations (from...import, import...as), understanding **name**, creating your own modules, and organizing code into files.
